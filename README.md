# Rajesh Power Admin

A modern, responsive administration portal for Rajesh Power Services, built with Next.js 16, React 19, and TypeScript.

## Included

- Secure branded login experience
- Responsive operations dashboard
- Collapsible vertical navigation
- Three horizontal submenu groups
- Search command panel
- Notifications and user profile menus
- Revenue, work-order, performance, and live-activity views
- Touch-friendly mobile layout with bottom navigation
- Responsive desktop, tablet, and phone breakpoints

## Run locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

The login screen is prefilled for demonstration. Selecting **Sign in to dashboard** opens the admin dashboard.

## Production check

```bash
npm run lint
npm run build
```
